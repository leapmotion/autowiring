Detailed Description

template<class T>
class auto_out< T >

Declares an output of an AutoFilter method via an argument.

Lazy allocation is implemented, and is triggered on destruction. Any attempt to access the referenced data will trigger instantiation and a commitment to output the data. Moved construction and assignment removes the option to output from the r-value operand, but does not force an output.

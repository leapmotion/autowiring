Invoked at the conclusion of a FindByTypeRecursive search operation

No information is passed about the success or failure of the search operation. Implementors of this interface are required to store enough context about the AnySharedPointer passed as the first argument to FindByTypeRecursive to determine the status of this operation, if that status is interesting.
